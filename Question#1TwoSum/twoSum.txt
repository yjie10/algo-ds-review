# Question: 
Given an array of integers, return the indicies of the two numbers that add up to a given target. 

Test Cases:
1. [1,3,7,9,2] target: 11 | [3,4];
2. [] target: 1 | null;
3. [1] target: 1 | null;
4. [1,3,7,9,2] target: 25 | null;
5. [1,6] target: 7 | [0,1];

# Analyzing Space and Time Complexity

Brute Force: 
Time: N * N = O(N^2)
Space: O(1)

Optimal:
Time: O(N)
Space: O(N)
